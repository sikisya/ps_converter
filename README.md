# PS Coverter

ネジコンをUSBで利用するためのコンバーター。

## 開発環境

|           名称          |バージョン|
|-------------------------|:-------:|
| Seeed SAMD Boards       |  1.8.2  |
| PsxNewLib               |  0.4.0  |
| Adfruit TinyUSB Library | 0.10.5  |

## オプション

- `USE_GAMEPAD_HAT_4_WAY`
    - 十字キーを4方向ハットスイッチとして扱う
- `USE_GAMEPAD_HAT_8_WAY`
    - 十字キーを8方向ハットスイッチとして扱う
- `未指定`
    - 十字キーをボタンとして扱う
